# Projeto-A3
Projeto A3 de Sistemas Distribuídos 

O projeto será um sistema de sestão de sedidos de E-commerce

Descrição: Criação de um sistema de e-commerce simples com micro-serviços para gerenciar pedidos, produtos e clientes. Os serviços serão divididos em micro-serviços 
(ex: gerenciamento de produtos, pedidos e usuários).


Será feito pelo VSCode, usando JAVA, HTML, JS, CSS e MySQL
